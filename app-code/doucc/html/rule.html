收益规则

1. 收益规则

购买一台AI机器人（1800元），每天收益36元。若扩展团队人数，则分为以下五个梯度：

1.团队人数10人以下（1级用户），收益为36+18*n（36为每日本金收益，18为团队每人增加的收益，n为拓展的团队人数，下同。）

例如邀请9位用户，则每天收益为36+18*9=198元，每月（30天，下同）收益5940元。

2.团队人数11-50人（2级用户），收益为36+（18+9）*n

例如邀请40位用户，则每天收益为36+（18+9）*40=1116元，每月收益33480元。

3.团队人数51-100人（3级用户），收益为36+（18+18）*n

例如邀请80位用户，则每天收益为36+（18+18）*80=2916元，每月收益87480元。

4.团队人数100-500人（4级用户），36+（18+18+9）*n（需团队中有2位达到二级用户并且超过25人）额外增加团队手续费分红15%。

例如邀请120位用户，则每天收益为36+（18+18+9）*120=5436元，加团队当天提现手续费15%的收益，比如团队当天提现手续费为20000元，则分红为20000*15%=3000元。

5.团队人数500以上（5级用户），36+（18+18+18）*n（需团队中有1位达到三级用户和2位二级用户）额外增加团队手续费分红30%

例如邀请520位用户，则每天收益为36+（18+18+18）*520=28116元，加团队当天提现手续费15%的收益，比如团队当天提现手续费为20000元，则分红为20000*30%=6000元。

2. 团队人数

团队人数按自己+一级邀请用户+二级邀请用户计算

一级邀请用户：用户注册填写自己邀请码注册的用户

二级邀请用户：用户注册填写一级用户邀请码注册的用户

3. 手续费规则

手续费分为本金手续费和收益手续费，本金手续费固定为1%。

收益手续费根据团队人数而变动：

1.团队人数50人以下，收益提现收取40%手续费（如果从购买产品之日算起满28天，则为30%）

2.团队人数50-100人，收益提现收取25%手续费

团队人数100人以上，收益提现收取20%手续费

4. 手续费分红

手续费分红有两个途径获得：

1.只要购买AI机器人就能获得分红，计算方式为每台AI可获得当天手续费0.5%的分红，例如5台AI，则分红为2.5%。满10台AI可获得当天手续费8%的分红（最高10台）。

2.是指根据自己的团队当天提现的手续费多少，分出固定的比例奖励给4级用户（15%）和5级用户（30%），比如自己的团队当天提现的手续费为20000元，则当天的分红收入为4级用户3000元，5级用户6000元。
